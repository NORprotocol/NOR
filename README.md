# NOR协议白皮书

## 介绍
NOR协议旨在通过整合先进的奖励系统、模型和再投资策略，彻底改变区块链生态系统的运作方式。我们的目标是为数字经济提供无缝、可扩展和高效的系统。

## 会员系统概述
会员系统是NOR协议的核心。它使用户能够根据他们对生态系统的贡献，获得独特的权益。
